Breakout! Hackvision Edition

This is a recreation of the classic game breakout. I made this game for the hackvision games console by Nootropic design but this game will work successfully on any arduino board with the TVout libary hardware and some push buttons. This game has been featured on the Nootropic design games page.

UPDATE! - v1.1 now features Wii nunchuk controles!

Download

If you feel the game is running too slow just edit the line “TV.delay_frame(2)” at the end to “TV.delay_frame(1)”

The following libraries are also required:

  • TVout Libary
  • Hackvision Controllers library
